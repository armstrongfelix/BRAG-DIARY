4. unddrstood th e concept attribute selector in css
        Attribute Selectors
        The attribute selectors allow you to target an attribute found on an HTML element.

        1. Links with "facebook" anywhere in the URL
                a[href*="facebook"] {
                color: #3C5A99;
                }
        
        2. Internal links, beginning with "#"
                a[href^="#"] {
                background-color: gold;
                }
        
        3. Links that end in ".org"
                a[href$=".org"] {
                color: red;
                }

animation:
    definition in words {
        A CSS animation is a sequence of visual changes applied to an element over a specified duration. 
        It allows for smooth transitions between different states of an element, such as changes in color, size, position, or opacity.
        CSS animations are defined using keyframes, which specify the styles at various points during the animation.
        The animation is controlled by properties such as duration, timing function, delay, iteration count, and direction.
    }
